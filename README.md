# Expense Tracker

## Description
A simple Python program that allows users to enter expenses and calculates the total amount spent.

## Features
- Add multiple expenses
- Calculates total spent
- Handles invalid inputs

## Technologies Used
- Python 3

## How to Run
1. Open terminal or command prompt.
2. Navigate to the project folder.
3. Run:

python expense_tracker.py

## Example
Enter expense amount: 100
Enter expense amount: 50
Enter expense amount: 20

Total Spent: ₹170
